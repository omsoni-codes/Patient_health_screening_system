# 🏥 Patient Health Screening System

A simple Python-based health screening program that evaluates a patient's basic health parameters and provides a quick risk assessment.

---

## 📌 Features

* 🧑‍⚕️ Classifies **age group** (Minor, Adult, Senior Citizen)
* ❤️ Analyzes **Blood Pressure (BP)** levels
* 🍬 Evaluates **Blood Sugar** condition
* ⚖️ Calculates **BMI category**
* 🚨 Generates an **Overall Health Risk Assessment**

---

## 🛠️ Technologies Used

* Python (Beginner-friendly)
* Console-based input/output

---

## 🚀 How It Works

The program takes the following inputs from the user:

* Patient Name
* Age
* Blood Pressure (BP)
* Blood Sugar Level
* Body Mass Index (BMI)

Based on these inputs, it classifies health conditions and prints a detailed report.

---

## ▶️ Example Output

```
Enter patient's name: Om
Enter patient's age: 20
Enter patient's BP: 135
Enter patient's sugar: 140
Enter patient's BMI: 24.5

--- Report for Om ---

Age Group: Adult (20)
BP: High Stage 1 (135)
Sugar: Diabetic (140)
BMI: Normal (24.5)

OVERALL: MEDIUM RISK - Monitor closely
```
